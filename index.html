1. Basic Linux File Commands

# i) List all contents of home directory with options
ls ~
ls -l ~
ls -a ~

# ii) Create a directory with your name
mkdir maddy

# iii) Create a blank file in that directory
touch maddy/file.txt

# iv) Copy and move the file
cp maddy/file.txt maddy/name2.txt   # copy file.txt to name2.txt
mv maddy/name2.txt maddy/name3.txt  # move/rename name2.txt to name3.txt

# v) Delete files and directory
rm maddy/file.txt
rm maddy/name3.txt
rmdir maddy

# vi) Show current working directory
pwd

2. Working with Text Files

# i)
touch f1.txt f2.txt

# ii)
cat >> f1.txt
# Enter: YourName RollNo SubjectName
# Ctrl + D to save

# iii)
cat >> f2.txt
# Enter: Department College City
# Ctrl + D

# iv)
cat f1.txt
cat f2.txt

# v)
cat f1.txt f2.txt > f3.txt

3. Manual Pages, Sorting, Piping

# i)
man ls > ls_manual.txt
man cat > cat_manual.txt
man sort > sort_manual.txt

# ii)
top              # Default

# iii)
sort file.txt
wc file.txt
grep "word" file.txt

# iv) Piping example:
ls -l | grep "txt" | wc -l

4. Vi Editor

vi file.txt
# Press i to insert
# Press ESC, then:
# :w to save
# :q to quit
# dd – delete line
# yy – copy line
# p – paste
# h,j,k,l – move left, down, up, right

5. File Permissions

touch filepermission.txt

# i) Letters
chmod u+rwx filepermission.txt
chmod g+rx filepermission.txt
chmod o+r filepermission.txt

# ii) Numbers (rwx = 7, rx = 5, r = 4)
chmod 754 filepermission.txt

10. Shell Script: Positive/Negative/Zero

#!/bin/bash
read -p "Enter a number: " num
if [ $num -gt 0 ]; then
    echo "Positive"
elif [ $num -lt 0 ]; then
    echo "Negative"
else
    echo "Zero"
fi

11. Shell Script: Digit Count (case)

#!/bin/bash
read -p "Enter a number: " num
length=${#num}
case $length in
1) echo "Single digit" ;;
2) echo "Double digit" ;;
3) echo "Three digit" ;;
4) echo "Four digit" ;;
*) echo "More than four digits" ;;
esac

12. Shell Script: Create Directory

#!/bin/bash
read -p "Enter directory name: " dir
if [ -d "$dir" ]; then
    echo "Directory exists"
else
    mkdir "$dir"
    echo "Directory created"
fi

13. Shell Script: Factorial (for loop)

#!/bin/bash
read -p "Enter a number: " n
fact=1
for ((i=1; i<=n; i++)); do
    fact=$((fact * i))
done
echo "Factorial: $fact"

14. Shell Script: Factorial (while loop)

#!/bin/bash
read -p "Enter a number: " n
fact=1
while [ $n -gt 1 ]; do
    fact=$((fact * n))
    n=$((n - 1))
done
echo "Factorial: $fact"

15. Shell Script: Login Details

Linux Viva Questions and Answers

1. Basic Linux File Commands

Q1: What does the pwd command do?

A: It prints the current working directory.

Q2: How do you list hidden files?

A: Use ls -a.

Q3: What's the difference between cp and mv?

A: cp copies the file; mv moves or renames it.

Q4: How do you delete a non-empty directory?

A: Use rm -r <directory>.

Q5: What happens if you use rm on a folder?

A: It gives an error unless you add the -r flag.

2. Text File Handling

Q1: What's the difference between cat > and cat >>?

A: > overwrites, >> appends to the file.

Q2: How do you combine two files?

A: cat file1 file2 > newfile.

Q3: Can cat create a new file?

A: Yes, with redirection (cat > filename).

Q4: Why use touch instead of cat?

A: touch creates empty files instantly without input.

3. Manual Pages and Piping

Q1: What does the man command do?

A: It displays manual pages for commands.

Q2: What does the top command show?

A: It displays real-time system processes and resource usage.

Q3: What is bottom?

A: A modern, user-friendly alternative to top.

Q4: How does grep work?

A: It searches for patterns in text.

Q5: What is piping in Linux?

A: Using | to pass output of one command to another.

4. Vi Editor

Q1: How do you enter insert mode in Vi?

A: Press i.

Q2: How do you save and exit Vi?

A: Use :wq.

Q3: What do dd, yy, and p do in Vi?

A: dd deletes line, yy copies line, p pastes.

Q4: How do you navigate without a mouse in Vi?

A: Use h, j, k, l for left, down, up, right.

5. File Permissions

Q1: What does chmod 754 mean?

A: User: rwx (7), Group: rx (5), Others: r (4).

Q2: How to remove read permission from group?

A: chmod g-r filename.

Q3: What is the difference between symbolic and numeric permission?

A: Symbolic uses letters (u, g, o); numeric uses numbers like 755.

6. User and Group Management

Q1: How do you add a new user?

A: sudo adduser username.

Q2: How do you change a user's password?

A: sudo passwd username.

Q3: How to add a user to a group?

A: sudo usermod -aG groupname username.

Q4: Difference between adduser and useradd?

A: adduser is user-friendly and interactive; useradd is low-level.

7. Synaptic Package Manager

Q1: What is Synaptic?

A: A GUI-based package manager for Debian-based systems.

Q2: How do you install Synaptic?

A: sudo apt install synaptic.

Q3: Can you install VLC via Synaptic?

A: Yes, search for VLC in Synaptic and install.

8. APT Package Manager

Q1: How do you install VLC using APT?

A: sudo apt install vlc.

Q2: What is the difference between remove and purge?

A: remove deletes software, purge also deletes config files.

Q3: What does apt update do?

A: Updates the list of available packages.

9. Network Commands

Q1: What does ifconfig do?

A: Displays network interface configuration.

Q2: What is ping used for?

A: Tests network connectivity.

Q3: What does netstat show?

A: Active connections and listening ports.

Q4: What does traceroute do?

A: Shows the path packets take to reach a destination.

10-14. Shell Scripts

Q1: What is the purpose of if-elif?

A: To check multiple conditions.

Q2: What does -gt, -lt, and -eq mean?

A: Greater than, less than, and equal to, respectively.

Q3: How do you check the length of a number?

A: Using ${#var}.

Q4: What is a case statement used for?

A: It simplifies checking multiple specific values.

Q5: How to check if a directory exists?

A: [ -d "dirname" ].

Q6: What is a for loop used for?

A: Repeating tasks a specific number of times.

Q7: Difference between while and for loop?

A: while repeats while a condition is true, for iterates over a range.

15-16. System Information

Q1: How do you list all users?

A: cut -d: -f1 /etc/passwd.

Q2: How do you check logged-in users?

A: Use who.

Q3: How to find your groups?

A: Use groups.

Q4: How to find current user?

A: Use whoami or $USER.

Q5: How to find OS type?

A: Use uname -o.

17-18. Servers

Q1: What is Apache?

A: An open-source web server.

Q2: How to check if Apache is running?

A: sudo systemctl status apache2.

Q3: What is Telnet used for?

A: Remote terminal access.

Q4: Is Telnet secure?

A: No, it sends data unencrypted.

Q5: What is the default port of Telnet?

A: Port 23.
